Funcionamiento.
El programa antes de entrar en ejecucion necesita un archivo sudo.txt que contenga unica y solamente la contraseña
encriptada(Usando el primer metodo del pdf) del usuario operador o administrador, esta a diferencia de las claves de los
usuarios comunes puede tener cualquier caracter para mas seguridad. Depues de haber creado el sudo.txt se requiere que
por lo menos se añada un usuario a la base de datos.

Nota: Se adjunta un sudo.txt como ejemplo. La contraseña desencriptada es 'AbCd'.
